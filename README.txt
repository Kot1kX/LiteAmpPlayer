# LiteAmp Player

**LiteAmp Player** es un reproductor de audio ligero para Windows, diseñado para reproducir música local de forma sencilla, rápida y sin depender de servicios externos.

El objetivo del proyecto es ofrecer un reproductor práctico, portable y sin distracciones: abrir música, reproducir, controlar volumen, usar amplificación moderada, repetir canciones o listas, activar aleatorio y aplicar perfiles básicos de ecualización.

## Características principales

* Reproducción de archivos de audio locales.
* Interfaz sencilla y directa.
* Lista de reproducción local.
* Control de volumen.
* Amplificador integrado.
* Limitador suave para reducir distorsión.
* Repetir canción.
* Repetir lista.
* Modo aleatorio.
* Ecualizador por presets.
* Funcionamiento local/offline.
* Sin cuentas.
* Sin anuncios.
* Sin nube.
* Sin telemetría.

## Privacidad

LiteAmp Player no recopila datos del usuario.

La aplicación:

* No envía archivos.
* No sube música a internet.
* No analiza hábitos de escucha.
* No comparte información con terceros.
* No utiliza servicios en la nube.
* No contiene telemetría propia.
* No requiere cuenta de usuario.
* No incluye publicidad.
* No ejecuta procesos de seguimiento en segundo plano.

Los archivos de audio se reproducen localmente en el equipo del usuario. La aplicación solo accede a los archivos o carpetas que el usuario selecciona manualmente.

## Seguridad y responsabilidad de uso

LiteAmp Player incluye control de volumen y amplificación. El uso de niveles altos de volumen, especialmente con auriculares, puede causar molestias, fatiga auditiva o daños en la audición.

El usuario es responsable de utilizar el reproductor con un volumen seguro.

Se recomienda:

* Empezar siempre con volumen bajo.
* Evitar usar amplificación alta con auriculares.
* No utilizar volumen máximo durante periodos prolongados.
* Reducir el volumen si aparece dolor, presión, pitidos o incomodidad.

El desarrollador no se hace responsable de daños auditivos, molestias físicas, pérdida de audición, daño en altavoces, auriculares u otros dispositivos causados por un uso irresponsable del volumen o del amplificador.

## Formatos compatibles

LiteAmp Player está pensado para reproducir formatos de audio comunes, incluyendo:

* MP3
* WAV
* FLAC
* AAC
* M4A

La compatibilidad puede depender también de los códecs disponibles en el sistema y del formato concreto del archivo.

## Cómo ejecutar LiteAmp Player

### Si has recibido una carpeta ya preparada

1. Abre la carpeta de LiteAmp Player.

2. Busca el archivo llamado:

   `LiteAmpPlayer.exe`

3. Haz doble clic sobre `LiteAmpPlayer.exe`.

4. El reproductor se abrirá y podrás añadir archivos o carpetas de música.

### Si has recibido el proyecto completo

El ejecutable normalmente se encuentra dentro de esta ruta:

`LiteAmpPlayer\bin\Release\net10.0-windows\LiteAmpPlayer.exe`

Para abrirlo:

1. Entra en la carpeta `LiteAmpPlayer`.
2. Abre la carpeta `bin`.
3. Abre la carpeta `Release`.
4. Abre la carpeta `net10.0-windows`.
5. Haz doble clic en `LiteAmpPlayer.exe`.

No es necesario introducir comandos para usar el reproductor si el ejecutable ya está compilado.

## Uso básico

1. Abre LiteAmp Player.
2. Pulsa **Añadir archivo** para seleccionar canciones concretas.
3. Pulsa **Añadir carpeta** para cargar música desde una carpeta.
4. Selecciona una pista de la lista.
5. Usa **Reproducir**, **Pausar**, **Detener**, **Anterior** o **Siguiente**.
6. Ajusta el volumen con el control correspondiente.
7. Usa el amplificador con prudencia.
8. Activa repetir canción, repetir lista o aleatorio si lo necesitas.
9. Selecciona un preset de EQ si quieres modificar ligeramente el sonido.

## Filosofía del proyecto

LiteAmp Player nace como una alternativa simple y local frente a reproductores pesados, con cuentas, anuncios, sincronización forzada o funciones innecesarias.

La idea es mantenerlo:

* Ligero.
* Local.
* Portable.
* Rápido.
* Sin dependencias innecesarias.
* Sin rastreo.
* Sin ruido visual.
* Sin enviar nada fuera del equipo.

## Aviso final

LiteAmp Player reproduce audio local y ofrece herramientas para modificar la salida de sonido. Usa el volumen y el amplificador con sentido común, especialmente si utilizas auriculares.

Tu música es tuya. Tus archivos se quedan en tu equipo. Tu volumen también es tu responsabilidad.
